## Quicure PDF 
### An offline, privacy-first PDF toolkit for Windows.

### Features
- 📄 Merge PDF — Combine multiple PDFs into one
- ✂️ Split PDF — Split PDF pages into individual files
- 📋 Extract PDF — Pull specific pages into a new PDF
- 🖼️ Image to PDF — Convert images into a PDF file
- 🔒 Encrypt PDF — Password-protect your PDF
- 🔓 Decrypt PDF — Remove password protection from a PDF

### Privacy
Everything runs 100% offline — your files never leave your device. An internet connection is only needed for license activation/subscription management.

### Requirements
Windows only.
